Python Restaurant Ordering System

Overview of the Project

This project is a simple Python-based restaurant ordering system that allows users to select food items from a predefined menu.
The program checks item availability, calculates the bill, and displays the total amount to be paid.

It is ideal for beginners learning Python basics such as:

Dictionaries

Conditionals

User input

Basic program flow

 Features

Displays a predefined restaurant menu with item prices.

Allows the user to order an item.

Option to add a second item.

Validates whether the ordered item exists in the menu.

Calculates and displays the total bill.

User-friendly text-based interface.

 Technologies / Tools Used

Python 3.x

Basic built-in functions: print(), input()

Dictionary for storing menu items and prices

Steps to Install & Run the Project
1. Install Python

Make sure Python 3 is installed on your system.
Check using:
python --version

2. Save the Script

Create a new file named restaurant.py and paste your code into it.

3. Run the File

Open a terminal/command prompt in the file location and run:
python restaurant.py
